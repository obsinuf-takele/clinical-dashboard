# Clinical Dashboard - Healthcare Management System

A comprehensive clinical dashboard with admin and doctor roles, patient management, appointment scheduling, and complete CRUD operations.

## Features

### Admin Features
- **Dashboard**: Overview statistics, clinic information, upcoming appointments
- **Doctor Management**: View, add, edit, delete doctor accounts
- **Patient Management**: View all patients, assign doctors
- **Complete CRUD Operations**: Create, Read, Update, Delete for both doctors and patients

### Doctor Features
- **Personalized Dashboard**: View assigned patients only
- **Patient Management**: Add, view, edit, delete own patients
- **Appointment Management**: View upcoming appointments
- **Patient Details**: Complete medical history and contact information

### Common Features
- **Secure Authentication**: Role-based login system
- **Search Functionality**: Filter doctors and patients in real-time
- **Responsive Design**: Works on desktop and mobile devices
- **Data Persistence**: Local storage for all data
- **Modern UI**: Gradient designs, animations, and intuitive interface
